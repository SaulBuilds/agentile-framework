## How to Update This Index

When a new module or crate is added to the project:

1. Create a `README.md` in the module's root directory using `templates/CRATE_README.template.md`
2. Add a row to the table above with the module name, path, README link, and status
3. Status values: `Current` (README matches code), `Stale` (needs update), `Missing` (no README)
