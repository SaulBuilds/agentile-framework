# Skip Protocol -- Experienced Engineer Fast Track

> **For qualified engineers who want to skip the onboarding quiz and start contributing immediately.**

## Overview

The Skip Protocol allows experienced developers to bypass the adaptive quiz and enter the Agentile framework at Journeyman tier (800 ELO). This is a fast track, not a free pass -- contributors who skip still must follow all core rules and earn Expert/Master through demonstrated work.

---

## Eligibility Criteria

A contributor qualifies if they meet **at least one** of the following:

### Path 1: Merged Pull Requests

- **3 or more merged PRs** in any public project in your domain
- PRs must be substantive (not typo fixes or dependency bumps)
- PRs must be verifiable via public links

### Path 2: Contribution History

- **50 or more contributions** visible on a public profile
- Quality matters more than count

### Path 3: Recognized Expertise

- Published author of relevant technical work
- Core maintainer of a significant open-source project
- Speaker at recognized technical conferences

---

## Process

### Step 1: Declaration

Provide evidence of eligibility (links to PRs, profile, published work).

### Step 2: Verification

An Expert+ contributor (or project maintainer if bootstrapping) reviews the evidence.

### Step 3: Agreement

The contributor explicitly agrees to:
1. Follow CORE_RULES.md
2. Follow the PR template
3. Follow the zooid's hard gates
4. Accept ELO scoring
5. Accept demotion if ELO drops below 800

### Step 4: Registration

Added to REGISTRY.md at 800 ELO, Journeyman tier.

### Step 5: Zooid Assignment

Choose any Journeyman-tier zooid based on background:
- Systems developer -> DEVELOPER
- Test/QA engineer -> QA_ENGINEER
- Technical writer -> TECH_WRITER

---

## What Skip Protocol Does NOT Grant

- **Expert tier**: Must be earned (1200+)
- **Master tier**: Must be earned (1600+)
- **ARCHITECT access**: Requires Expert (1200+)
- **FORMAL_VERIFIER access**: Requires Master (1600+)
- **Self-merge privileges**: All Journeyman PRs require Expert+ review
- **Exemption from hard gates**: All gates apply to everyone

---

## FAQ

**Q: Can I start at Expert or Master?**
A: No. Maximum starting ELO is 1000 (quiz) or 800 (skip). Higher tiers are earned.

**Q: Can an AI agent use the Skip Protocol?**
A: Yes, with verifiable evidence of prior contributions.

**Q: What if my evidence is in a private repository?**
A: A project maintainer with access can verify, or provide redacted screenshots.
