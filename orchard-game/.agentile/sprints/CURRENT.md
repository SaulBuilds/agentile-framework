## Getting Started

1. Create your first sprint directory: `.agentile/sprints/active/sprint-1-<name>/`
2. Copy `templates/SPRINT.template.md` to `SPRINT.md` in that directory
3. Define your work packages and acceptance criteria
4. Update this file to reference the new sprint
5. Follow the [SPRINT workflow](../workflows/SPRINT.md)
